# Smart Parking Project 

## Installation Instructions
1. **Clone the repository**:  
   Open your terminal and run the following command to clone the repository:
   ```bash
   git clone https://github.com/thnhngaan/NT131_SmartParkingProject.git
   cd NT131_SmartParkingProject
   ```

2. **Install dependencies**:  
   Ensure you have Python and pip installed. Then install the required packages from `requirements.txt`:  
   ```bash
   pip install -r requirements.txt
   ```

## Project Structure
- **MAIN.py**: Entry point for the application to start the smart parking system.
- **models/**: Directory containing all the necessary model files that handle data and operations relevant to parking management.
- **input/**: Directory which holds input files needed for the system to function properly. Make sure to place your input files here for the application to process them.

## Usage Instructions
1. After setting up your environment and installing the necessary packages, navigate to the project directory. 
2. Run the application by executing:
   ```bash
   python MAIN.py
   ```
3. Follow the console prompts to use the smart parking features such as checking availability, reserving spots, and more.
